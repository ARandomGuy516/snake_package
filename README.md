# snake

A tiny terminal snake game, importable as a Python package.

## Install

From GitHub with `uv`:

```bash
uv add git+https://github.com/ARandomGuy516/snake_package
```

Or with `pip`:

```bash
pip install git+https://github.com/ARandomGuy516/snake_package
```

## Play

```bash
snake
# or
python -m snake
```

## Use as a library

```python
from snake import Snake, main

main()  # run the game
```
