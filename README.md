# gameshell

A library for making classic terminal games easily. Currently includes snake.

## Install

From GitHub with `uv`:

```bash
uv add git+https://github.com/ARandomGuy516/gameshell_package.git
```

Or with `pip`:

```bash
pip install git+https://github.com/ARandomGuy516/gameshell_package.git
```

## Play

```bash
snake
# or
python -m gameshell
```

## Use as a library

```python
from snake import main

main()  # run the game
```
